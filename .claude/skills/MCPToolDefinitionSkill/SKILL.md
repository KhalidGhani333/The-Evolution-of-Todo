Skill Name: MCPToolDefinitionSkill
Purpose:
Define and implement MCP tools following Official MCP SDK patterns.

Capabilities:
Create tool schemas with proper typing
Define parameter validation rules
Implement stateless tool handlers
Connect tools to database operations
Return standardized response formats
Document tool usage and examples

Usage Rules:
Used by MCP-Server-Builder agent
All tools must be stateless
Must include user_id parameter for data isolation
Follow Official MCP SDK conventions
Each tool must have clear input/output contracts
